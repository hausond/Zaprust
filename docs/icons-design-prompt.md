# Промпт для дизайн-агента — иконки приложения Zaprust

> Скопируй этот файл целиком в дизайн-агент. На выходе нужны **отдельные файлы** с точными именами и форматами из таблицы в конце. Ничего не объединяй в спрайт-лист, каждый значок — свой файл.

## Контекст

**Zaprust** — лёгкое нативное Windows-приложение (десктоп, Rust + egui). Тёмная плоская минималистичная тема. Окно фиксированного размера ~420×560. Сейчас в интерфейсе используются юникод-символы вместо иконок, и они не отображаются — нужно заменить их растровыми иконками. Также у приложения нет собственной иконки (в таскбаре и заголовке окна — заглушка).

Палитра приложения (для справки и для цветной иконки-логотипа):

| Назначение | HEX |
|---|---|
| Акцент (зелёный, «работает/старт») | `#388E3C` (и светлее `#2EA043`) |
| Опасное (красный, «стоп») | `#C64040` |
| Фон панели | `#181A1E` |
| Поле/кнопка | `#202328` |
| Очень тёмный фон (лог) | `#121316` |
| Приглушённый серый (текст/иконки) | `#60646C` |
| Белый текст | `#FFFFFF` |

## Общий стиль ВСЕХ интерфейсных иконок (кроме логотипа)

- **Стиль:** современные **линейные (outline)** иконки, минималистичные, геометричные. Исключение — «play» и «stop»: они **залитые** (filled), как принято для медиа-кнопок.
- **Цвет:** **чистый белый `#FFFFFF`** на **полностью прозрачном** фоне. (Приложение само перекрашивает их в нужный цвет темы — поэтому строго одноцветные, без градиентов и теней.)
- **Холст:** **64×64 px**, прозрачный (RGBA).
- **Поле артворка:** рисунок по центру, в пределах центральных **~52×52 px** (поля ~6 px со всех сторон), оптический центр.
- **Толщина обводки:** ~**5 px** при холсте 64×64 (визуально ~2 px при отображении в 24 px). **Скруглённые** концы и соединения (round cap / round join).
- **Единообразие:** одинаковая визуальная «масса» и толщина линий у всего набора, пиксель-снап для чёткости.
- **Без фона, без подложки, без рамок, без теней.**

## Список интерфейсных иконок (обязательные)

1. **Старт (play)** — залитый треугольник «play», направлен вправо, со слегка скруглёнными углами. Белый. → `icon-play.png`
2. **Стоп (stop)** — залитый квадрат со скруглёнными углами. Белый. → `icon-stop.png`
3. **Скачать (download)** — стрелка вниз, упирающаяся в горизонтальную черту/«полку» (классический символ загрузки). Outline. → `icon-download.png`
4. **Обновить / пересканировать (refresh)** — круговая стрелка (одна, по часовой, с наконечником). Outline. → `icon-refresh.png`
5. **Настройки (gear)** — шестерёнка (6–8 зубцов) с отверстием в центре. Outline. → `icon-settings.png`
6. **Назад (back)** — стрелка влево (←), outline, скруглённый наконечник. → `icon-back.png`
7. **Шеврон вниз (chevron-down)** — «‹v›», галочка-шеврон вниз (для «развернуть/расширенный режим»). Только обводка. → `icon-chevron-down.png`
8. **Шеврон вверх (chevron-up)** — «‹^›», шеврон вверх (для «свернуть/простой режим»). → `icon-chevron-up.png`

## Список интерфейсных иконок (опциональные, но желательны)

9. **Отмена (cancel/X)** — крестик «×» из двух линий, outline, скруглённые концы. → `icon-cancel.png`
10. **Тест (test)** — стилизованный глобус/сеть или галочка-в-кружке (символ проверки доступности). Outline. → `icon-test.png`
11. **Ожидание (busy)** — песочные часы или круг-спиннер (статичный кадр). Outline. → `icon-busy.png`

## Логотип / иконка приложения (цветная, отдельный стиль)

Это «лицо» приложения — **щит** (символ защиты/обхода блокировок). Может быть цветным и чуть детальнее интерфейсных иконок.

- **Форма:** скруглённый геральдический щит. Заливка — акцентный зелёный (`#388E3C` → `#2EA043`, допустим лёгкий вертикальный градиент). Внутри — белый минималистичный эмблематичный мотив: на выбор стилизованная буква **«Z»**, либо «пробивающая» стрелка/разрыв (намёк на обход DPI), либо замочная скважина. Выбери самый чистый и узнаваемый вариант в маленьком размере.
- **Фон:** прозрачный (для PNG). Для `.ico` — тоже прозрачный фон.
- **Читаемость в 16–24 px обязательна** (это размер в таскбаре/заголовке): без мелких деталей, толстые формы, высокий контраст эмблемы на щите.
- **Поставки логотипа:**
  - `icon-app-256.png` — 256×256, цветной, прозрачный фон.
  - `icon-app-64.png` — 64×64, тот же логотип (для шапки окна рядом с названием).
  - `icon-app.ico` — мультиразмерный Windows-icon, кадры **16, 24, 32, 48, 64, 128, 256** px, прозрачный фон (для иконки exe/таскбара).

## Требования к выводу (важно)

- **Формат интерфейсных иконок:** PNG-24 с альфа-каналом (RGBA), фон прозрачный, размер ровно **64×64**.
- **Имена файлов — точно как в таблице** (латиница, нижний регистр, дефисы, расширение `.png`/`.ico`). Без пробелов и кириллицы в именах.
- **Каждая иконка — отдельный файл.** Никаких спрайт-листов и контактных листов.
- Дополнительно (по желанию) — отдать **SVG-исходники** интерфейсных иконок теми же именами с расширением `.svg` (масштабируемые мастера). Но PNG обязательны.
- Прозрачность настоящая (premultiplied не нужно), без белой/чёрной подложки по краям, без обрезки артворка холстом.

### Памятка (для разработчика, после получения файлов)
Интерфейсные иконки белые на прозрачном — грузятся как egui-текстуры и **перекрашиваются в коде** (tint) под цвет элемента (белый на кнопках, серый `#60646C` для шевронов/назад). Логотип и `.ico` идут как есть. Окно/таскбар: PNG → `egui::IconData` через `with_icon`, exe-ресурс — из `icon-app.ico` на шаге упаковки.
