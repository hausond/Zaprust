Zaprust — лёгкий GUI для обхода DPI-блокировок (поверх zapret/Flowseal)
======================================================================

КАК ПОЛЬЗОВАТЬСЯ
1. Распакуйте папку Zaprust в путь БЕЗ пробелов, кириллицы и спецсимволов.
   Хорошо:  C:\Zaprust\
   Плохо:   C:\Users\Иван Петров\Загрузки\Zaprust (новый)\
2. Запустите zaprust.exe (двойной клик).
3. Если ядра нет — нажмите «Скачать ядро» (подтянется актуальная сборка Flowseal).
4. Нажмите «Старт». Появится запрос прав администратора (UAC) — нажмите «Да».
   Это нужно один раз для установки службы обхода; дальше обход поднимается сам
   при включении компьютера.
5. Простой режим сам подберёт рабочую стратегию. В расширенном режиме можно
   выбрать стратегию вручную, отредактировать списки (⚙) и обновить ядро.

ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
- В расширенном режиме нажмите «Логи» (откроется папка logs рядом с exe) и
  пришлите файл zaprust.log — в нём вся картина.
- Либо нажмите «Диагностика» — краткая сводка скопируется в буфер обмена,
  её можно сразу вставить в чат.

ВОЗМОЖНЫЕ ПРЕДУПРЕЖДЕНИЯ СИСТЕМЫ
- SmartScreen на неподписанном exe: «Подробнее» → «Выполнить в любом случае».
- Антивирус может ругаться на WinDivert (он же используется любым zapret) —
  это ложное срабатывание (PUA/RiskTool). Добавьте папку Zaprust в исключения.
- Рекомендуется включить в браузере Secure DNS (DoH).

УДАЛЕНИЕ
- Перед удалением папки нажмите в расширенном режиме «Удалить службу» —
  это остановит и удалит службу обхода и выгрузит драйвер.
- После этого можно просто удалить папку Zaprust целиком.

СОСТАВ ПАПКИ
  zaprust.exe   — само приложение (один файл, без дополнительных DLL)
  logs\         — логи (создаются при первом запуске)
  core\         — ядро zapret (если отсутствует — кнопка «Скачать ядро»)

Zaprust — независимый сторонний GUI, не аффилирован с bol-van или Flowseal.
